2. Padronização das Cores dos Cards de Preço

Situação Atual

Os cards de preço estão com cores inconsistentes pelo site. Alguns são azul/ciano, outros são rosa.
Alterações Necessárias

Padronizar TODOS os cards de preço para a cor rosa (#F03D87 )
Garantir que a cor rosa seja mantida tanto no estado normal quanto no hover/clique
Aplicar esta padronização em todos os produtos do site
Código CSS para Padronização das Cores

css
/* Estilo base para todos os botões de preço */
.preco-btn, 
.tamanho-btn, 
.comprar-btn,
button[class*="preco"],
button[class*="tamanho"],
button[class*="comprar"] {
  background-color: #F03D87 !important; /* Rosa-chiclete */
  color: white !important;
  border: none !important;
  transition: all 0.3s ease;
}

/* Estilo para hover e active (quando clicado) */
.preco-btn:hover, 
.tamanho-btn:hover, 
.comprar-btn:hover,
button[class*="preco"]:hover,
button[class*="tamanho"]:hover,
button[class*="comprar"]:hover,
.preco-btn:active, 
.tamanho-btn:active, 
.comprar-btn:active,
button[class*="preco"]:active,
button[class*="tamanho"]:active,
button[class*="comprar"]:active,
.preco-btn:focus, 
.tamanho-btn:focus, 
.comprar-btn:focus,
button[class*="preco"]:focus,
button[class*="tamanho"]:focus,
button[class*="comprar"]:focus {
  background-color: #F03D87 !important; /* Mantém rosa mesmo no hover/active */
  opacity: 0.9; /* Apenas reduz levemente a opacidade para feedback visual */
  color: white !important;
  box-shadow: 0 0 8px rgba(240, 61, 135, 0.5) !important; /* Adiciona um sutil efeito de brilho */
}

/* Classe específica para botões rosa (caso seja necessário aplicar diretamente) */
.rosa {
  background-color: #F03D87 !important;
  color: white !important;
}
