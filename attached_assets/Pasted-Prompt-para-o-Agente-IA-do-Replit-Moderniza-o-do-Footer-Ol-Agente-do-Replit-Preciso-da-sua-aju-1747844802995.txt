Prompt para o Agente IA do Replit: Modernização do Footer

Olá, Agente do Replit! Preciso da sua ajuda para modernizar o footer do site da confeitaria ÉDoce, tornando-o mais atrativo, moderno e responsivo. Abaixo estão as instruções detalhadas para essa atualização.
Objetivo

Transformar o footer atual em um design mais moderno, com ícones para redes sociais (Instagram e WhatsApp), melhor organização visual e garantir que seja totalmente responsivo em todos os dispositivos.
Elementos a Manter

Logo da ÉDoce
Slogan "Adoçando momentos especiais"
Links rápidos para as páginas principais
Informações de contato (WhatsApp e Instagram)
Horário de funcionamento
Informações de copyright
Melhorias Solicitadas

1. Design e Layout

Adotar um layout em grid com 2-3 colunas em desktop e 1 coluna em dispositivos móveis
Aumentar o espaçamento entre elementos para melhor legibilidade
Adicionar uma linha sutil de separação entre o conteúdo principal e o footer
Utilizar cores que combinem com a paleta atual do site (tons de rosa, turquesa)
Implementar transições suaves nos elementos interativos
2. Ícones de Redes Sociais

Adicionar ícones modernos para Instagram e WhatsApp (preferencialmente SVG)
Posicionar os ícones de forma destacada, possivelmente em uma área separada
Implementar efeito hover nos ícones (mudança de cor, leve aumento de tamanho)
Garantir que os ícones sejam clicáveis e redirecionem para os perfis/contatos corretos
3. Responsividade

Garantir que o footer se adapte perfeitamente a todos os tamanhos de tela
Em dispositivos móveis, reorganizar os elementos em uma única coluna centralizada
Ajustar tamanho de fontes e ícones para melhor visualização em diferentes dispositivos
Implementar media queries para controlar o layout em diferentes breakpoints
4. Elementos Adicionais (Opcionais)

Adicionar um botão "Voltar ao topo" discreto
Incluir um pequeno formulário de newsletter (se aplicável)
Adicionar animações sutis nos elementos do footer

Implementação de Ícones

Para os ícones, recomendo utilizar uma das seguintes opções:
Font Awesome - Adicionar no head do HTML:
html
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.0.0/css/all.min.css">
E substituir as classes de ícones por:
icon-instagram → fab fa-instagram
icon-whatsapp → fab fa-whatsapp
icon-clock → far fa-clock
Bootstrap Icons - Adicionar no head do HTML:
html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.0/font/bootstrap-icons.css">
E substituir as classes de ícones por:
icon-instagram → bi bi-instagram
icon-whatsapp → bi bi-whatsapp
icon-clock → bi bi-clock

Considerações Finais

Mantenha a consistência com o restante do site em termos de cores, fontes e estilo
Teste o footer em diferentes tamanhos de tela para garantir a responsividade
Verifique se todos os links estão funcionando corretamente
Certifique-se de que os ícones de redes sociais são facilmente reconhecíveis
Garanta que o contraste entre texto e fundo seja adequado para boa legibilidade
Por favor, implemente essas melhorias no footer do site, mantendo a identidade visual da confeitaria ÉDoce, mas trazendo um visual mais moderno, atrativo e responsivo.
